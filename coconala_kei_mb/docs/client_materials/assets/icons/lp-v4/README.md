## 2) 命名ルール（推奨）
「どのセクションの何のアイコンか」が一目で分かる命名にしてください。

形式:
- `s{番号}_{用途}_{連番}.{拡張子}`

例:
- `s3_step_01.svg`
- `s3_step_02.svg`
- `s3_step_03.svg`
- `s7_faq_01.svg`

補足:
- 英小文字 + 数字 + `_` のみ
- スペース/日本語/記号は使わない
- 可能なら `svg` 優先、写真系は `webp`/`png`
